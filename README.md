# PokeAPI를 이용한 포켓몬 도감 만들기
-------------------------------------

### 개발환경
- Javascript
- HTML
- CSS

### 프로젝트 기간

2024.08.14 ~ 2024.08.21


### 기능


1. https://pokeapi.co/api/v2/pokemon/ Open API 사용, 포켓몬의 정보 받아오기
  
2. 포켓몬의 번호, 이름, 사진 표시하고 클릭하면 그 포켓몬의 상세정보를 노출

3. 검색창을 만들어 특정 문자를 검색 했을 때 특정 문자를 포함한 포켓몬들을 노출한다.
