# PokeAPI를 이용한 포켓몬 도감 만들기
-------------------------------------

### 개발환경
- Javascript
- HTML
- CSS



### 기능


1. https://pokeapi.co/api/v2/pokemon/ 사용하여 API 통신 후 포켓몬 정보 노출 (https://pokeapi.co/ API 사용)
  
2. 포켓몬의 번호, 이름, 사진 표시하고 클릭하면 그 포켓몬의 상세정보를 노출한다.

3. 검색창을 만들어 특정 문자를 검색 했을 때 특정 문자를 포함한 포켓몬들을 노출한다.
